## 🚀 Getting Started

### Prerequisites

- **Python 3.x** — [Download here](https://www.python.org/downloads/)
- **Jupyter Notebook** or **JupyterLab**
- Optional: `matplotlib`, `graphviz` (used in some notebooks)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/<your-username>/data-structures-in-python.git
   cd data-structures-in-python
   ```

2. **Install Jupyter** (if not already installed)

   ```bash
   pip install jupyter
   ```

3. **Install optional dependencies**

   ```bash
   pip install matplotlib graphviz
   ```

4. **Launch Jupyter Notebook**

   ```bash
   jupyter notebook
   ```

5. Open any `.ipynb` file and start exploring! 🎉
